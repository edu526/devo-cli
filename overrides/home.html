Features

Everything you need

Built for teams that move fast

Database Connect

Run devo ssm connect to open a secure port-forwarding tunnel to any private RDS, ECS or internal service via AWS SSM — no VPN, no bastion SSH.

AWS SSO Login

Run devo aws-login to authenticate all your AWS profiles at once and track credential expiry.

DynamoDB Export

Run devo dynamodb export to export any table to CSV, JSON, JSONL or TSV with filtering — no console needed.

Auto Commit Messages

Generate conventional commit messages from your staged changes automatically with devo commit. No context switching.

Code Review

Automated security analysis and best-practice checks on your branch with devo code-reviewer before it hits the PR.

Standalone Binaries

Single binary for Linux, macOS and Windows. Self-updating with devo upgrade.
